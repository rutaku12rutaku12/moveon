## 👨‍💻 프로젝트 소개
지역 정착 과정에서 발생하는 불편과 생활정보 공백을 해소하고, 안전하고 편리한 정착을 지원하는 것을 핵심가치로 삼는 **생활 편의 통합 플랫폼**입니다.

공공 데이터 API와 카카오맵 API를 연동하여 신뢰성 있고 최신화된 생활 정보를 제공하는 것을 목표로 하였습니다.

## 🕰 개발 내용
### 개발기간
2025-08-21 ~ 2025-09-12
### 기술스택
HTML, CSS, JavaScript, Java, JSP, Spring Boot, MySQL, GitHub 등
### 주요기능
* 현재 위치 정보를 기반으로 기온, 습도, 풍속 등의 날씨 정보 표시
* MySQL과 위치 정보 기반으로 쓰레기 배출 관련 정보 표시
* 공공 데이터를 활용하여 지도에 의류수거함의 위치 표시
* 현재 위치를 기준으로 특정 반경 내 성범죄자 인원수 등의 정보 표시
* 지하철 엘리베이터/에스컬레이터의 위치 정보 및 운행 상태 정보 표시
